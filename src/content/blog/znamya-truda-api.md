## О клиенте

**ФК «Знамя Труда»** — старейший действующий футбольный клуб России. Основан в 1909 году. Финалист Кубка СССР 1962 года (играли с донецким «Шахтёром» перед 102 000 зрителей в Лужниках). 4-кратный чемпион Москвы 1910–1913.

Сейчас клуб играет в Первенстве Федерации футбола Московской области — это любительский уровень. Но история и бренд — серьёзные.

## Задача

Нужен был сайт с:

- Разделами: Главная, История, Команда, Клуб, Матчи, Новости, Медиа.
- Лендингом для продажи членства (с онлайн-оплатой через Т-Кассу).
- **Актуальной статистикой**: турнирная таблица, расписание, бомбардиры, составы.
- Админкой, чтобы клуб сам обновлял контент.

Бюджет ограниченный (клуб содержит себя сам, без спонсорских миллионов).

## Что я сделал

**Стек:** Astro + Tailwind, хостинг GitHub Pages, DNS Cloudflare (без прокси), админка на Decap CMS.

### Самое интересное — API Федерации

Покопавшись в коде сайта `ffmo.ru`, я обнаружил, что там современный API Platform с документацией по Hydra (`https://ffmo.ru/api/docs.jsonld`). Эндпоинты:

- `/api/teams` — команды.
- `/api/matches` — матчи.
- `/api/standings` — таблица.
- `/api/tournaments` — турниры.
- `/api/tournament_top_players` — бомбардиры.
- `/api/players` — игроки.

API закрытое — для использования нужен токен партнёра. Написал клубу текст обращения в ФФМО: «мы клуб-участник, дайте доступ к API для публикации статистики на собственном сайте». Обычно федерации такие запросы одобряют.

### Пока ждём токен — Google Sheets

На время получения доступа — ведение через Google-таблицу. Один человек обновляет таблицу после тура, сайт через 15 минут показывает свежие данные. Не идеально, но работает.

Когда токен придёт — переключаю источник данных одной строкой в `src/content/data.ts`.

## Интеграции

- **Т-Касса** — онлайн-оплата членства (карта, СБП, Tinkoff Pay, SberPay). Пока на паузе: клубу нужно подключить интернет-эквайринг в Т-Банке.
- **Timepad** — продажа билетов на встречи с командой и мероприятия. Виджет вставлен на сайт, клуб сам создаёт события.
- **Яндекс.Карты** — интерактивная карта стадиона (вместо статичного изображения — виджет).

## Что уникального в этом проекте

- **Бесплатный хостинг навсегда.** Сайт такой историчной организации работает на $0 в месяц.
- **Статистика в реальном времени из официального источника.** Не «руками забитые числа», а данные Федерации.
- **Админка без технаря.** Любой человек в клубе может обновить состав или добавить новость.

## Итог

**Цена:** 27 500 ₽.
**Срок:** 1,5 недели.
**Что вошло:** сайт + лендинг членства + админка + Т-Касса + Timepad + Google Sheets для статистики + настройка домена.

Клуб получил то, что на рынке стоит 150–300 тысяч. Я получил кейс с настоящей историей — и публичное одобрение от «Клуба-старожила российского футбола».
